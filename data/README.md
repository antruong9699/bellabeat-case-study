## Dataset

**FitBit Fitness Tracker Data** — CC0 Public Domain

Download from Kaggle: https://www.kaggle.com/datasets/arashnic/fitbit

After downloading, unzip and place the CSV files in this folder.
The following files are used in this analysis:
- dailyActivity_merged.csv
- heartrate_seconds_merged.csv
- hourlyCalories_merged.csv
- hourlyIntensities_merged.csv
- hourlySteps_merged.csv
- minuteSleep_merged.csv
- weightLogInfo_merged.csv
